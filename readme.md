# FizzBuzz Code Kata

This is a FizzBuzz code kata starter project. Read about the kata here: https://kata-log.rocks/fizz-buzz-kata

## Project Structure

- **FizzBuzz**: `FizzBuzz/FizzBuzz.cs` is the class to further extend.
- **FizzBuzzTests**: `FizzBuzzTests/FizzBuzzTests.cs` contains one test to get started.

## .NET version

This project uses .NET 10. Instead of installing .NET 10 you can also switch to an older version by adjusting the `<TargetFramework>` property in `Directory.Build.props`:

```xml
<Project>
  <PropertyGroup>
    <TargetFramework>net10.0</TargetFramework>
    ...
  </PropertyGroup>
</Project>
```
